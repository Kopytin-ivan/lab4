# Предсказание шероховатости обработанной поверхности по шуму во время фрезерования

## Тема проекта

Проект посвящён анализу данных, полученных во время процесса фрезерования.  
Основная задача — предсказать шероховатость обработанной поверхности на основе акустического сигнала, параметров подачи, частоты вращения шпинделя и глубины резания.

## Цель проекта

Разработать функциональный прототип приложения/аналитического решения, которое использует data-driven подход и методы машинного обучения для прогнозирования качества обработки поверхности.

## Объект анализа

Производственный процесс фрезерования металлической заготовки.

## Используемые данные

Используется датасет Milling Surface Roughness Acoustic Sensor Dataset.

Датасет содержит аудиофайлы, записанные во время фрезерования, а также таблицу со значениями шероховатости поверхности для разных режимов обработки.

## Целевая переменная

Шероховатость обработанной поверхности.

## Признаки

В проекте планируется использовать следующие признаки:

- признаки, извлечённые из акустического сигнала;
- скорость вращения шпинделя;
- скорость подачи;
- глубина резания;
- статистические и спектральные характеристики сигнала.

## Планируемые модели машинного обучения

В проекте будут рассмотрены модели регрессии:

- Linear Regression;
- Random Forest Regressor;
- Gradient Boosting Regressor.

## Аналитические задачи

В рамках проекта планируется:

- предсказать шероховатость поверхности;
- определить, при каких режимах обработки качество ухудшается;
- найти пороговые значения акустических признаков, после которых шероховатость выходит за допустимый уровень;
- подобрать оптимальные режимы обработки для получения заданной чистоты поверхности.
